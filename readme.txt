For more info visit the [plugin homepage](http://www.cognitivegenerationenterprises.com/)

**Methods**

This plugin let's you insert code any a wide variety of ways.

*Shortcodes:* Create your own custom shortcodes and put them anywhere you want - even in comments!

*Widgets:* Create a custom widget that shows up right on the Widgets screen.

*Hooks:* You can also use any of our code hooks to place code in the:

1. Header (between 'head' tags)
2. Footer (bottom of the page)
3. Before page content (before main WP loop)
4. After page content (after main WP loop)
5. Before post content
6. After post excerpt (after 'more' tag)
6. After post content

*Manual:* This lets you place a simple function directly into your theme files that can call much more complex blocks of code.

**Display Filters**

* Insert on pages - here you can select pages where you code CAN BE shown. 
* Exclude pages - pages where you code CAN NOT be shown. 
* Hide from user - allows to hide code from specified user groups/roles. 

**Other options:**

* Priority number - determines display order of the Inserts placed in the same location.
* Align - allows you to set horizontal align for inserted code. Useful for aligning ads and other visual content.

== Installation ==

1. Check that you meet the minimal system requirements: WordPress 3.1, PHP 5.2.4
2. Follow the standard WordPress plugin installation procedure
3. Activate the plugin through the Plugins menu in WordPress
